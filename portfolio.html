🚀 Overview

This is my personal developer portfolio — a fully responsive, minimal site built entirely with HTML and CSS, designed to showcase my skills, projects, and personality. I built this from scratch with a strong focus on accessibility, mobile-first design, and semantic HTML — without relying on JavaScript frameworks or any pre-built templates.

The goal: create a site that is clean, professional, and easy to navigate across all devices, while staying lightweight and performant.

⚙️ Tech Stack & Highlights

  • HTML5 – Semantic markup across all sections
  • CSS3 – Global, modular styles with responsive layout
  • Media Queries – Adaptive scaling across mobile and desktop
  • Google Fonts – Modern typography with Quicksand & La Belle Aurore
  • Formspree – For functional contact form submissions
  • GitHub Pages – Deployed via GitHub Actions on every push to master

✅ Features

  • 📱 Mobile-First Layout
    Optimized for all screen sizes, with a custom landscape-only layout on mobile to maintain desktop-like consistency.
  • 🖼️ Project Showcase Pages
    Dedicated HTML pages for each project with a description, tech stack, and live/demo links.
  • 📬 Functional Contact Form
    Real submission powered by Formspree (with spam protection notice).
  • 🌐 Global Styles + Orientation Prompts
    A central global.css handles zoom, orientation warnings, and consistent layout.
  • 🔒 Minimal JavaScript Usage
    No JS dependencies — just one optional script to toggle contact details.

🧪 Testing & Deployment

  • Manual Testing across Chrome, Firefox, Safari
  • Viewports: 1024px+, 768px, 480px (landscape + portrait)
  • Automated Deployment via GitHub Actions on every commit to master

✨ Unique Touches

  • Custom mobile landscape prompt
  • Accessibility-aware layout with zoomable UI
  • Personal branding through clean icons and consistent typography
  • No third-party templates — everything authored from scratch

💡 Key Learnings

  • Designing mobile-first for real-world usability
  • Deploying static sites with GitHub Pages + GitHub Actions
  • Structuring and maintaining global CSS across multiple HTML pages
  • Prioritizing clarity, accessibility, and minimalism in frontend development

🌟 Final Outcome

This portfolio serves as a professional, lightweight, and fully custom representation of who I am as a developer. It allows recruiters, collaborators, and visitors to:

  • Learn about me and my background
  • Browse my key projects
  • Reach out via a functional contact form
  • Get a clear sense of my attention to detail and frontend sensibility
